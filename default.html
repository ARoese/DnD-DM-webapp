Help

Commands

add: add and effect. Usage: [add FRIENDLY_NAME TIME]

remove: remove an effect. Usage: [remove FRIENDLY_NAME]

exit: stops server back-end. Usage: don't

You can click any element in the list to remove it

Time Format

Time format when entering commands: days:hours:minutes:seconds Any values that are zero before the first non-zero value can be ommitted. ex: 1:24 instead of 0:0:1:24

while negative numbers are sometimes supported, functionality is not guaranteed. Using negative numbers in the progress command signifies going back in time
